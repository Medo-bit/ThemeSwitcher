# ThemeSwitcher

A PowerShell-based application that dynamically adjusts the theme of Windows 10/11 between light and dark modes based on the sunrise and sunset times. The application operates online for data retrieval and offline for the remainder of its functionality, ensuring uninterrupted operation regardless of environmental conditions and without incurring excessive internet traffic. It utilizes a modern context menu for seamless navigation and an enhanced user experience without compromising system performance.
